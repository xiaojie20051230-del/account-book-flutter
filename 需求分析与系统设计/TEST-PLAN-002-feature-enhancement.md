## 2. 白盒测试（自动化）

### 2.1 CSV 导出

| 测试 | 说明 |
|------|------|
| `CsvExporter 生成 CSV 格式正确` | 验证 CSV 头和行为符合预期 |
| `CsvExporter 空数据不崩溃` | 无账单时导出，生成正确空 CSV |
| `CsvExporter 备注含逗号时转义` | 备注含逗号时用引号包裹 |

### 2.2 统计图表

| 测试 | 说明 |
|------|------|
| `StatsPage 无数据时显示"暂无数据"` | 当月无账单时显示空状态 |
| `StatsPage 饼图只计算支出` | 收入不进入饼图 |
| `StatsPage 柱状图显示 12 个月` | 全年每月都有柱状条目 |

### 2.3 搜索筛选

| 测试 | 说明 |
|------|------|
| `search 按备注关键字过滤` | 关键字匹配时返回正确结果 |
| `search 关键字为空返回全部` | 搜索框清空后恢复显示 |
| `search 不区分大小写` | "Lunch" 和 "lunch" 都匹配 |

### 2.4 预算管理

| 测试 | 说明 |
|------|------|
| `monthlyBudget 默认值为 0` | 未设置预算时返回 0 |
| `预算进度条低于 100% 时绿色` | 支出小于预算时进度条为绿色 |
| `预算进度条超过 90% 时红色` | 支出接近预算时变红 |

### 2.5 深色模式

| 测试 | 说明 |
|------|------|
| `ThemeProvider 默认浅色` | 初始状态为 light |
| `toggle 切换主题` | 调用后从 light 变为 dark |
| `toggle 再次切换回到 light` | 调用两次后回到 light |

### 2.6 数据备份/恢复

| 测试 | 说明 |
|------|------|
| `BackupManager 生成 .abk 文件` | 备份文件包含所有 Hive 数据 |
| `BackupManager 恢复后数据一致` | 恢复后数据与备份时一致 |

### 2.7 滑动删除

| 测试 | 说明 |
|------|------|
| `Dismissible 滑动触发确认` | 滑动后回调正确执行 |
| `撤销删除后数据恢复` | SnackBar 撤销按钮恢复数据 |

## 3. 黑盒测试（手动）
