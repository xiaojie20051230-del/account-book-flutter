# 测试记录

随手记项目测试记录目录。

---

## 文件结构

| 文件 | 内容 |
|------|------|
| `REPORT-001-mvp.md` | MVP 阶段测试报告（TC-01 ~ TC-08、E-1 ~ E-6） |
| `REPORT-002-features.md` | 功能扩展测试报告（CSV导出、图表、搜索、预算、深色） |
| `REPORT-003-enhancements.md` | 功能增强测试报告（备份、收入分类、滑动删除） |
| `REPORT-004-trash.md` | 回收站与修改追踪测试报告 |
| `BUG-LOG.md` | 所有 Bug 汇总，持续更新 |
| `screenshots/` | 测试截图 |

---

## 格式规范

- 每个 REPORT 记录一个阶段的测试结果
- 所有用例通过时只写总结，不列明细
- 失败的用例才在报告中记录
- Bug 统一汇总到 `BUG-LOG.md`
