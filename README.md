# 随手记 — Flutter 记账 App

创新实验 期末综合项目（第 3 组）

## 项目概述

基于 Flutter 的跨平台记账应用，接口驱动架构，纯本地存储，支持低端设备。

## 技术栈

- **框架：** Flutter 3.44 / Dart 3.12
- **状态管理：** Provider
- **本地存储：** Hive
- **图表：** fl_chart
